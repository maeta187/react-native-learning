# 📘 React Native 学習手順（実務経験者向け）

## ✅ 前提条件

- React / TypeScript 実務経験あり
- React Native / Expo は未経験
- コードは **TypeScript**
- 開発環境は **Expo（最新）**
- 公式ドキュメントを参照して学習

---

## 🔰 ステップ 1: React Native の基礎を理解する

### 📚 公式ドキュメント

- [React Native Docs](https://reactnative.dev/docs/getting-started)

### 🎯 学習目標

- `View`, `Text`, `StyleSheet`, `Button` などの基本コンポーネント
- JSX の文法と Web との違い
- フレックスボックスによるレイアウト

---

## 🚀 ステップ 2: Expo の基礎を理解する

### 📚 公式ドキュメント

- [Expo Docs](https://docs.expo.dev/)

### 🎯 学習目標

- Expo CLI のインストールと初期設定
- `expo start` による開発環境の立ち上げ
- 実機／シミュレーターでの確認方法

---

## 🛠 ステップ 3: 簡単なアプリを作ってみる

### 🧪 アプリ例：ToDo リスト

- `expo init` を使って TypeScript テンプレートで作成
- `useState` と `FlatList` を使ったタスクの表示／追加

### 🎯 学習目標

- props / state / イベントの扱い方
- コンポーネント分割と再利用
- Expo の開発環境に慣れる

---

## 🔧 ステップ 4: 機能拡張しながら理解を深める

### 💡 機能追加例

- React Navigation による画面遷移
- AsyncStorage によるデータの永続化
- Expo Camera / ImagePicker によるメディア操作

### 🎯 学習目標

- ライブラリの導入と活用
- 状態管理の高度化（Context API や Zustand など）
- モバイル特有の UI / UX に対応

---

## 🎓 ステップ 5: 最終アプリを完成させる

### 🏁 ゴール

- 自作アプリを構想・実装・改善
- `expo build` または `eas build` でアプリをビルド
- 実機で動作確認＆配布体験

---

## 🔍 まとめ

| 項目         | ゴール                                                     |
| ------------ | ---------------------------------------------------------- |
| React Native | 基本コンポーネントと構文理解                               |
| Expo         | 開発・デバッグ・ビルド環境の理解                           |
| アプリ制作   | 設計〜実装〜配布までの一連の開発体験                       |
| 問題解決能力 | ドキュメントを活用して自力で実装・解決できるようになること |
